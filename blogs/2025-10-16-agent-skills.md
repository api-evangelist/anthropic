Agent Skills are organized folders of instructions, scripts, and resources that Claude loads dynamically to perform specialized tasks. The launch includes Anthropic-managed Skills for PowerPoint, Excel, Word, and PDF, plus a Skills API for uploading custom skills. Skills use progressive disclosure — metadata loads at startup, instructions when triggered, resources on demand — so a workspace can install many skills without context penalty.
