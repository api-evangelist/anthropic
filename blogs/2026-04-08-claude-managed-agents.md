Claude Managed Agents is a fully managed agent harness for running Claude as an autonomous agent with secure sandboxing, built-in tools, and SSE streaming. New Agents, Sessions, and Environments APIs let you define versioned agent configurations, configure cloud containers, and run stateful sessions. All endpoints require the managed-agents-2026-04-01 beta header.
