Anthropic launched Claude Platform on AWS, bringing the Claude API to Anthropic-managed infrastructure accessible through AWS, with AWS billing and IAM authentication. The full Messages API, Files API, Message Batches API, Claude Managed Agents, Agent Skills, code execution, and tool use are all accessible through native AWS endpoints. The new platform sits alongside Microsoft Foundry as an Anthropic-operated cloud option distinct from partner-operated Bedrock and Vertex AI.
