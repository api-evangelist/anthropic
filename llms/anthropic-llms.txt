# Anthropic — Claude API

> Anthropic is an AI safety company and creator of the Claude family of large language models (Opus, Sonnet, Haiku, and the Fable/Mythos frontier line). The Claude API provides text generation, vision, tool use, extended/adaptive thinking, batch processing, and agentic workflows including managed agents, skills, memory, compaction, and computer use. Anthropic also authors the open Model Context Protocol (MCP) and ships Claude Code, the terminal agentic coding tool.

This file is generated by API Evangelist from the catalog. Anthropic also publishes a first-party llms.txt at https://platform.claude.com/docs/llms.txt.

## APIs
- [Messages API](https://docs.anthropic.com/en/api/messages): Generate the next message in a conversation with text, images, tool use, thinking, streaming, and structured outputs.
- [Models API](https://docs.anthropic.com/en/api/models-list): List and inspect Claude models and their capabilities.
- [Message Batches API](https://docs.anthropic.com/en/api/creating-message-batches): Submit batches of message requests at a 50% discount.
- [Files API](https://docs.anthropic.com/en/api/files-create): Upload and reuse files across Messages, Batches, and Agents.
- [Admin API](https://docs.anthropic.com/en/api/admin-api/users/get-user): Manage org, members, invites, workspaces, and API keys.
- [Prompt Tools API](https://docs.anthropic.com/en/api/prompt-tools-generate): Generate, improve, and templatize prompts.
- [Token Counting API](https://docs.anthropic.com/en/api/messages-count-tokens): Count tokens without creating a message.
- [Skills API](https://docs.anthropic.com/en/api/skills/create-skill): Create and manage custom Agent Skills (beta).
- [Usage & Cost API](https://docs.anthropic.com/en/api/usage-cost-api): Programmatic usage and cost reporting.
- [Claude Code Analytics API](https://docs.anthropic.com/en/api/claude-code-analytics-api): Daily Claude Code productivity metrics.
- [Managed Agents API](https://platform.claude.com/docs/en/managed-agents/overview): Agents, Sessions, Environments with SSE streaming (beta).
